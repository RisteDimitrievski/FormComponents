# FormComponents Set of form components that make your front end work easy. Compatible with Bootstrap 3. Form Components allow you easy and fast build html forms. @version 1.0 @Author Riste Dimitrievski Required Laravel version: 8.x If you know the XenForo Framework you will easy understand how this components work. :) Currently available tags: accepts everything as standard html tag do. You can specify also label and hint properties (description to the input) example: The accepts 3 properties (name, label, hint) and HTML attributes also. The require you to specify keys in keys property (each key should be comma separated) and options data should be also in comma separated. Example how it will work: The option1 is parent of array1 key. The tag accepts also HTML standard attributes like standard tag
